Source downloads

OpenClover repositories. They contain sources of OpenClover (since version 4.2.0). Please raise pull requests to these repositories.
  • https://github.com/openclover/clover - Clover Core + Ant + Eclipse + IDEA
  • https://github.com/openclover/clover-maven-plugin - Clover for Maven
  • https://github.com/openclover/grails-clover-plugin - Clover for Grails
  • https://github.com/openclover/clover-aspectj-compiler - Clover for AspectJ
  • https://github.com/openclover/clover-examples - code samples
Atlassian Clover® repositories. They contain open source version of Atlassian Clover® 4.1.2.
  • https://bitbucket.org/atlassian/clover - Clover Core + Ant + Eclipse + IDEA
  • https://bitbucket.org/atlassian/maven-clover2-plugin - Clover for Maven
  • https://bitbucket.org/atlassian/grails-clover-plugin - Clover for Grails
  • https://bitbucket.org/atlassian/clover-examples - code samples
Other repositories. Plugins released after 11 April 2017 integrate with OpenClover.
  • https://github.com/jenkinsci/clover-plugin - Jenkins Clover Plugin
  • https://github.com/hudson2-plugins/clover-plugin - Hudson2 Clover Plugin
  • https://github.com/hudson3-plugins/clover-plugin - Hudson3 Clover Plugin
Looking for Bamboo Clover Plugin sources? You must have a commercial Bamboo license to get them from Atlassian.
