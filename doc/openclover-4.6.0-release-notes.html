OpenClover 4.6.0 Release Notes

2024-03-31
We're excited to share with you OpenClover 4.6!

Changes

Features

Official support for Java 17

In releases 4.5.0 to 4.5.2, which introduced experimental support for JDK17, we ironed out all missing Java language features, special cases and bugs. Now we are thrilled to announce official support for Java 17! You can instrument and compile code with any language level between 8 and 17 and run it with OpenClover in JDK/JRE 8 to 17.

Dropped support for Java 1.7

In preparation for upcoming support for Java 21, we're dropping support for Java 1.7 in this release. While you can still use the "1.7" language level in OpenClover's configuration, it will be treated as Java 8, and a warning will be printed. All OpenClover classes have been compiled with Java 8 target, thus JDK/JRE 8+ is required to run the code anyway. While this is a breaking change, current Java 1.7 popularity in the world is marginal, thus we decided to ship it in the minor release.

Full support for module-info.java

The project Jigsaw introduced modules in Java 9. OpenClover ignored this fact for a pretty long time. In this release OpenClover contains module-info information for all its components (OpenClover Runtime, OpenClover Core, OpenClover for Ant, OpenClover for Maven) in their JAR files. What's more, OpenClover will now automatically instrument module-info.java files, adding package imports for OpenClover Runtime classes.

Support for Groovy 3.x and 4.x

A smaller brother of Java, Groovy language, finally got some love. We've rewritten Groovy's abstract syntax tree transformations as well as updated libraries, to ensure compatibility with Groovy 3 and 4.

Groovy 2.x is no longer unsupported

As Groovy 2.x is end of life, OpenClover is no longer compatible with Groovy 2.

Upgrading from OpenClover 4.5.x

Clover for Ant

Simply download new Clover-for-Ant zip file and use the clover.jar from it.

Clover for Eclipse

Download new update site ZIP and install in your Eclipse.

Clover for IDEA

Download new plugin JAR and install in your IDEA.

Clover Maven Plugin

Bump version number to 4.6.0 for 'org.openclover:clover-maven-plugin' and 'org.openclover:clover' in your pom.xml files.

Jenkins Clover Plugin

Upgrade to the latest 4.15.x plugin version.
