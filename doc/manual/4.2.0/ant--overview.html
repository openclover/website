Clover-for-Ant

What is Clover for Ant?

Clover-for-Ant integrates the industry-leading code coverage tool, Clover with the Apache Ant build automation tool. Clover-for-Ant allows you to easily measure the coverage of your unit tests, enabling targeted work in unit testing — resulting in stability and enhanced quality code with maximal efficiency of effort.
