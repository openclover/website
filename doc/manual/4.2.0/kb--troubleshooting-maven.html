Troubleshooting Maven

You are reading documentation for version 4.2.x. Check the latest version.

This page describes issues specific for Clover-for-Maven (2 & 3) plug-in. 

 

  • Could not find goal 'setup' in plugin org.apache.maven.plugins:maven-clover-plugin:2.4
  • Duplicate class errors with Clover and JAXB or JAXB2 plugin
  • Duplicate class errors with Clover and jaxws-maven plugin
  • Maven 2 plugin fails with cannot find symbol exception when defining two locations for source files
  • Maven is deploying instrumented jars
  • Warning about older Clovered artifact version during multi-module Maven build
