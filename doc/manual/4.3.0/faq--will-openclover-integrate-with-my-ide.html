You are reading documentation for version 4.3.x. Check the latest version.

Q: Will Clover integrate with my IDE?


Clover provides an integrated plugin for Eclipse and IDEA.Clover should also work happily with any IDE that provides integration with the Ant, Maven or Grails build tool.
