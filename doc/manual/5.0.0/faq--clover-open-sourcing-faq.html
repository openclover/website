Clover open-sourcing FAQ

Atlassian Clover is now available as an open-source project. Effective April 11, 2017 Clover is no longer available for purchase or renewal. 

Read the official announcement here: https://www.atlassian.com/blog/announcements/atlassian-clover-open-source

In short: no license is needed

OpenClover requires no license of any kind. It is free and open source, with no license tiers, no license keys and no usage limits. The project is developed at https://github.com/openclover/clover and released to Maven Central under the org.openclover group ID - there is no need to clone and compile it yourself. See Downloads.

Any advice mentioning -Dmaven.clover.licenseLocation or a clover.license file is obsolete and can be ignored.
