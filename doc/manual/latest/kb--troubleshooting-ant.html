Troubleshooting Ant

This page describes problems specific for the Clover-for-Ant plug-in. 

 

  • A <files> with <includesfile> or <excludesfile> ignores patterns
  • Clover.db is not generated at location specified by initString
  • Error reading Clover 4.0.3 JAR file
  • Error reading project file on <clover-env/>
  • Properties such as Excludes is not being passed on to antcall tasks
  • Using external javac compiler causes classes to not be instrumented with Clover
