Troubleshooting Compilation

This page describes problems related with Java or Groovy source code instrumentation by Clover and further compilation using javac or groovyc. It covers Clover integrations with build tools (such as Ant, Maven or Grails) as well as standalone instrumentation (using the CloverInstr command line tool).

  • Compilation error with Clover enabled - unexpected return value
  • Compilation error with Clover enabled - variable of type Object
  • Java 8 code instrumented by Clover fails to compile
  • Java- Scala cross-compilation error - cannot find symbol
  • Method code too large
