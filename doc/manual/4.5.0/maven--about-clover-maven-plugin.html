About Clover-for-Maven

Overview

The Clover-for-Maven plugin allows you to produce Clover code coverage reports from the Maven 2 and 3 build tools. It provides detailed information to highlight areas of low coverage in your project, helping to guide your unit-testing activities.

Open Source status

This plugin is open source, under the Apache license. See Developer Guide.

Support

Please report any issues with the Clover-for-Maven plugin in the OpenClover issue tracker.

Acknowledgments

The Maven Clover plugin is derived from the original Maven plugin for Clover written by Vincent Massol. The groupId has been changed from org.apache.maven.plugins to org.openclover.
