Clover-for-IDEA

What is Clover-for-IDEA?

Clover-for-IDEA brings the industry-leading code coverage tool, Clover, to the IntelliJ IDEA integrated development environment. Clover-for-IDEA allows you to easily measure the coverage of your unit tests, enabling targeted work in unit testing resulting in stability and enhanced quality code with maximal efficiency of effort.
