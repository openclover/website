Welcome to the Clover Knowledge Base

Troubleshooting Ant

This page describes problems specific for the Clover-for-Ant plug-in.

Troubleshooting Maven

This page describes issues specific for Clover-for-Maven (2 & 3) plug-in.

Troubleshooting Grails

This page describes problems specific for the Clover-for-Grails plugin.

Troubleshooting Compilation

This page describes problems related with Java or Groovy source code instrumentation by Clover and further compilation using javac or groovyc. It covers Clover integrations with build tools (such as Ant, Maven or Grails) as well as standalone instrumentation (using the CloverInstr command line tool).

Troubleshooting Execution

This page describes problems related with execution of an application or tests instrumented by Clover, including but not limited to: managing coverage database and recording files, setting up correct runtime class path or Xbootclasspath, solving NoClassDefFoundError or VerifyError messages.

Troubleshooting Reports

This page describes problems related with generation of Clover's HTML, XML, PDF or JSON reports, including but not limited to: lack of coverage data, lack of per-test coverage, wrong classification of application vs test classes, rendering source code view.

Troubleshooting Performance

This page describes issues related with long-running compilation, testing or report generation as well as out-of-memory errors.

Troubleshooting Bamboo

This page describes problems related with Clover integration with Bamboo using Bamboo Clover Plugin. Integration can be either automatic (Ant-, Maven- or Grails-based builds) or manual (any build system).
