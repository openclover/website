OpenClover is free, open-source, and always will be

Your donation keeps it that way

OpenClover is maintained by volunteers in their spare time. Every feature added, every bug fixed, and every compatibility update for a new Java or build tool release is unpaid work driven by passion for code quality. If OpenClover saves you time or improves the quality of your software, please consider giving something back.

I have been maintaining OpenClover as a volunteer for nearly a decade. I genuinely love this project, but love alone does not pay the bills. Running OpenClover costs real time and money every year:

  • My time — design, coding, code reviews, issue triage, user support, documentation
  • Website hosting — keeping openclover.org up and fast for everyone
  • IntelliJ IDEA — the IDE used to develop OpenClover itself
  • Claude Code Pro — an AI coding assistant that has recently become an important part of my development workflow, helping me move faster

None of this has ever been sponsored or reimbursed. Every cent has come from my own pocket because I believe OpenClover is worth it. If you do too, a small donation helps me keep the lights on and spend more of my spare time writing code rather than worrying about costs.

More value from better coverage data

Smarter test suites

Accurate, per-test coverage reports reveal redundant tests and coverage gaps. Teams that act on this data routinely reduce test count by 10–30% while improving overall coverage — cutting both CI time and maintenance burden.

Bugs caught earlier

Uncovered code is where bugs hide. Research consistently shows that a bug found during coding costs 5–10× less to fix than the same bug discovered in production. OpenClover makes uncovered code impossible to ignore.

Confidence to refactor

A live coverage map gives teams the confidence to refactor aggressively, knowing exactly which paths are exercised and which are not. Better code structure means lower long-term maintenance cost.

The community maths

OpenClover has thousands of active users, based on downloads statistics. If just 1 in 10 bought a single cup of coffee once a year, that would be enough to sponsor one full-time engineer dedicated to OpenClover development. Imagine what that engineer could build for the whole community.
